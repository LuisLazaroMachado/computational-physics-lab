# Computational Physics Lab

Interactive computational physics simulations built with **Python** and **Streamlit**. This project provides visual and interactive demonstrations of fundamental physics concepts, making them easier to understand through numerical simulation and graphical visualization.

## Simulations

### Hydraulic Force
Calculates the hydrostatic force exerted by water on a vertical dam by allowing users to modify physical parameters and observe the resulting force.

### Electric Field
Visualizes the electric field generated by a uniformly charged rod, helping users explore how charge distribution affects the surrounding field.

### Standing Waves
Simulates standing wave formation on a stretched string, allowing experimentation with parameters such as frequency and tension.

### Optical Mirage
Illustrates the optical mirage phenomenon by simulating how light rays bend due to temperature gradients in the atmosphere.

## Installation

```bash
git clone https://github.com/LuisLazaroMachado/computational-physics-lab.git
cd computational-physics-lab
pip install -r requirements.txt
```

## Running the Application

```bash
streamlit run app.py
```

or

```bash
python -m streamlit run app.py
```
